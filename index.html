نظام إدارة صالون الحلاقة - خطط الأسعار

المميزات الأساسية (متوفرة في جميع الخطط):

  • إدارة الخدمات: إضافة / تعديل / حذف الأسعار.
  • تسجيل خدمات الزبائن وحساب الفاتورة.
  • طباعة الفاتورة (PDF أو طباعة مباشرة).

المميزات الإضافية (متوفرة في بعض الخطط):

  • لوحة تحكم للإيرادات اليومية والأسبوعية والشهرية.
  • إدارة المستخدمين (إضافة موظفين - حسابات دخول).
  • تقارير الإيرادات (Excel / PDF).

خطط الأسعار:

● 1) الخطة المقطوعة (One-time Payment):
  • دفع لمرة واحدة وامتلاك النظام بالكامل.
  • استضافة مجانية لمدة سنة.
  • دعم فني مجاني لأول شهر.
  • جميع الشاشات الأساسية + الإضافية.
  • السعر: 6000 جنيه مصري
● 2) الخطة الشهرية (Basic Subscription):
  • بدون دفع مقدم.
  • اشتراك شهري ثابت.
  • يشمل الشاشات الأساسية فقط (بدون Dashboard أو إدارة المستخدمين).
  • دعم شهري محدود (2 تعديل شهريًا فقط).
  • السعر: 200 جنيه شهريًا
●3) الخطة المرنة (Hybrid Plan):
  • تدفع مرة واحدة للتجهيز الأولي (1000 جنيه).
  • ثم اشتراك شهري منخفض (150 جنيه/شهر).
  • يشمل الشاشات الأساسية + Dashboard.
  • دعم شهري موسع + تحديثات مجانية.
  • السعر: 1000 جنيه أول مرة + 150 جنيه مصري شهرياً
